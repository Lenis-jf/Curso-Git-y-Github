GIT & GIT HUB

Aquí estan algunos comandos y terminos para manejar mejor Git y Github. Espero que estes preparado, lo que se viene es GENIAL!

COMANDOS
  • git init: Inicia el repositorio en un directorio

  • git add: Añade archivos a Staging

  • git commit: Envia los ultimos cambios al repositorio

  • git status: muestra el estado de la BD

  • git show: Muestra los cambios mas recientes

  • git log: Muestra el historial de cambios de un repositorio
  • git config --global: Permite cambiar la configuración

  • git diff: Muestra las diferencias entre commits

  • git checkout: Nos permite navegar entre commits y ramas

  • git rm: Elimina archivos

  • git reset: Otra manera de eliminar archivos

  • git branch: Crea ramas
